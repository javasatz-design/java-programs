Program 1: Reverse a String

String: "Java is Fun"

Expected Output: nuF si avaJ

Answer:

                String s1 = "Java is Fun";
                String result = "";
                for (int i = s1.length() - 1; i >= 0; i--) {
                    result += s1.charAt(i);
                }
                System.out.println("The value is: " + result);

Program 2: Find Factorial of a Number

Number: 5

Expected Output: 120

Answer:

                int num = 5;
                int result = 1;
                for (int i = 1; i <= num; i++) {
                    result *= i;
                }
                System.out.println("Factorial: " + result);
