Program 1: Reverse a String

String: "Java is Fun"

Expected Output: nuF si avaJ

Answer:


                String str = "Java is Fun";
                String result = "";
                for (int i = str.length() - 1; i >= 0; i--) {
                    result += str.charAt(i);
                }
                System.out.println("The value is: " + result);

Program 2: Reverse only alphabets in a given string

String: 1ab23

Expected Output: 1ba23

Answer:


                String str = "1ab23";
                int left = 0;  
                int right = str.length() - 1; 

                char[] strCharArr = str.toCharArray();

                while(left < right) {
                    if (!Character.isLetter(strCharArr[left])) {
                        left++;
                    } else if (!Character.isLetter(strCharArr[right])) {
                        right--;
                    } else {
                        char tempvar = strCharArr[left];
                        strCharArr[left] = strCharArr[right];
                        strCharArr[right] = tempvar;
                        left++;
                        right--;
                    }
                }

                System.out.println("The reverse value is " + new String(strCharArr));
